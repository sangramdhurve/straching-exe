DROP MIXAMO ANIMATIONS HERE
===========================
From mixamo.com -> Animations tab -> pick a stretch/warm-up -> Download:
  Format: FBX Binary
  Skin:   With Skin
  FPS:    30
  In Place: ON (if available)

Save each as <name>.fbx in this folder. I import, frame, and render a looping clip.
Good search terms: stretch, warming up, neck, quad, hamstring, side bend, reach.
