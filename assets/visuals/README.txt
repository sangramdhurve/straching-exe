STRETCH VISUALS
===============

Drop one image per stretch here, named exactly after the stretch `id`
(see assets/data/stretches.json), e.g.:

    neck_side_stretch.png
    wall_calf.png
    chair_hamstring.png

The app loads `assets/visuals/<id>.png` automatically. Until a file exists,
a friendly "Visual coming soon" placeholder is shown — so the app runs fine
with no images yet.

Recommended (launch): clean 2D illustrated poses, ~1080x1080 PNG.
Later upgrade: short looping MP4 (then set the stretch's assetType to "video").
